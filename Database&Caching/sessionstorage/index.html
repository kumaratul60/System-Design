Tab Duplication & Context Isolation Sandbox Replication Gotchas

Test the isolation boundaries of page sessions by trying the following steps:

Experiment Checklist:
1. Add a custom key-value pair in this tab.
2. Click the Duplicate Tab Programmatically button below.
3. Inspect the duplicated tab: the keys are copied! (Copy-on-write).
4. Change/add a key in the new tab, then go back to the original tab: **they do not sync!** (Decoupled contexts).
5. Copy the URL and paste it in a brand new, manually opened tab: **it starts completely empty!**
