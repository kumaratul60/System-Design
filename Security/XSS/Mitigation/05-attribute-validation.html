Attribute & Protocol Validation

Don't just escape characters—ensure the data makes sense (e.g., a link should start with http).

Visit User Link (Validated)

Why this works:

Attackers often use the javascript: protocol to run code when a user clicks a link. By checking the beginning of the string for http://, we block those scripts entirely.

Try this: Append ?url=javascript:alert('evil') to the URL. The application will detect the dangerous protocol and refuse to update the link.
