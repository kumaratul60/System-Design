Content Security Policy (CSP)

CSP is your safety net. Even if you have an XSS bug, the browser will refuse to run the injected script.

Why this works:

The browser reads the CSP meta tag (or header) and becomes very strict. If an attacker tries to inject <script>alert(1)</script>, the browser will say: "This script is not from a trusted source" and block it.

Check the Console: Append ?q=<img src=x onerror=alert(1)>. You will see a "Refused to execute script" or "Refused to run the JavaScript URL" error in the console. The alert will NOT fire.
