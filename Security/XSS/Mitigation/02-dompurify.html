HTML Sanitization with DOMPurify

Sometimes you NEED to show HTML (like a rich-text editor). Use a sanitizer to strip the "poison" out.

Safe HTML Output:

Why this works:

DOMPurify understands the HTML spec. It allows "safe" tags like <b> but instantly deletes dangerous ones like <script> or attributes like onerror.

Try this: Append ?html=<b>Hello</b><script>alert('evil')</script> to the URL. You will see "Hello" in bold, but the script will be gone.
